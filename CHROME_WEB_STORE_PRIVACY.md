# Chrome Web Store 隐私权规范填写内容

完整发布填写稿见 [CHROME_WEB_STORE_SUBMISSION.md](./CHROME_WEB_STORE_SUBMISSION.md)。本文件保留隐私权规范标签页的核心字段，便于直接复制。

以下内容用于 Chrome Web Store 开发者后台的“隐私权规范”标签页，可按字段复制填写。

## 单一用途说明

薇译 Vanslate 的单一用途是：在用户主动触发或用户配置的自动规则命中时，使用用户自行配置的 OpenAI 兼容大模型 API，对当前网页中的可见文本、文章导航文本、虚拟滚动中新出现的文本或用户选中的文本进行 AI 上下文翻译与表达处理，并在原网页上下文中展示结果。

扩展的相关功能，包括全文翻译、划词解释、总结、改写、术语表、本地缓存、失败重试、自动翻译规则、目标语言跳过、悬浮工具和快捷设置，均服务于这一网页上下文理解、翻译和改写用途。

## 远程代码使用理由

本扩展不加载、执行或注入远程 JavaScript、远程 WASM 或其他远程可执行代码。扩展的 JavaScript、CSS、HTML 和图标资源均打包在扩展包内。

扩展会向用户在设置页自行填写的 OpenAI 兼容 API 地址发送 HTTPS 请求，以获取翻译、解释、总结、改写或其他用户配置工具的输出。该请求只用于远程模型推理，不会把远程响应作为代码执行。模型返回内容会作为文本或 JSON 数据解析后显示，不作为脚本运行。

## 主机权限使用理由

需要 `<all_urls>` 主机权限，是因为用户希望在任意网页上使用全文翻译、划词翻译、右键翻译和自动翻译规则。扩展需要在用户访问的不同网站中读取可见文本内容，才能识别待翻译文本并把译文插入到原文附近。

扩展不会后台抓取用户浏览历史，不会在未触发翻译功能时上传网页内容。翻译请求只在用户点击翻译、使用右键菜单、划词翻译，或用户配置的自动翻译规则匹配当前 URL 时发生。

## `activeTab` 权限使用理由

需要 `activeTab` 权限，是为了在用户点击扩展按钮或使用当前标签页相关操作时，临时访问当前活动标签页并触发翻译、清除译文或划词翻译。

该权限只用于用户当前正在操作的标签页，不用于持续访问其他标签页内容。

## `contextMenus` 权限使用理由

需要 `contextMenus` 权限，是为了在浏览器右键菜单中提供“薇译：翻译全文”入口，让用户可以在当前网页上直接通过右键菜单触发全文翻译。

右键菜单只作为用户主动触发翻译的交互入口，不用于收集额外数据。

## `scripting` 权限使用理由

需要 `scripting` 权限，是为了在用户主动触发翻译功能时，向当前网页注入扩展自带的内容脚本和样式文件。内容脚本负责识别页面中的可见文本块、显示加载状态、插入译文、显示悬浮按钮和处理虚拟滚动中新出现的内容。

注入的脚本和样式均来自扩展包内部，不从远程服务器加载。

## `storage` 权限使用理由

需要 `storage` 权限，是为了在浏览器本地保存用户配置和本地缓存，包括：

- API 地址、API Key、模型列表和当前模型。
- 源语言、目标语言、译文渲染方式、译文 CSS 和悬浮工具显示方式。
- 自动翻译规则、术语表、失败重试次数。
- 页面处理提示词、划词工具提示词、单工具模型覆盖配置、本地缓存和缓存清理状态。

这些数据保存在用户浏览器的扩展本地存储中。扩展不运营中转服务器，不把这些设置上传给扩展作者控制的服务器。API Key 只在向用户自行配置的模型 API 发起翻译请求时作为认证信息使用。

## 数据使用情况确认

确认：薇译 Vanslate 的数据使用情况符合 Chrome Web Store 开发者计划政策和用户数据政策。

扩展处理的数据仅用于提供用户请求的网页翻译、划词解释、总结、改写、术语表、本地缓存、自动翻译规则和相关设置功能。扩展不会出售用户数据，不会将用户数据用于广告投放、信用评估、数据经纪、与网页上下文理解和表达无关的用户画像分析，或任何与扩展单一用途无关的目的。

翻译时，扩展会将待翻译文本、源语言、目标语言、术语表和模型参数发送到用户自行配置的 OpenAI 兼容 API 服务商。该服务商可能是 OpenAI、用户自建服务或其他第三方供应商。除用户配置的模型 API 服务商外，扩展不会向其他第三方分享用户数据。

## 可补充到隐私政策中的摘要

薇译 Vanslate 不提供内置模型服务。用户在设置中自行填写 API 地址、API Key 和模型名称。扩展仅在用户主动触发全文翻译、划词工具、右键翻译，或用户配置的自动翻译规则匹配当前网页时，读取当前网页可见文本或用户选中文本，并发送到用户配置的模型 API 以获得翻译、解释、总结或改写结果。扩展设置和翻译缓存保存在浏览器本地。扩展不收集遥测，不出售用户数据，不注入广告，不运营中转服务器。
