# 更新日志

## v1.1.0 - 2026-05-12

- 发布版本升级到 `1.1.0`，Chrome 打包产物改为 `vanslate-1.1.0.zip` 和 `vanslate-1.1.0.crx`。
- 打包脚本改为自动读取 `package.json` 版本，并复用稳定 `.pem` 密钥，降低后续发布出错概率。
- 产品页增加更新日志入口，并同步下载链接到 `vanslate-1.1.0.crx`。
- 补齐面向开源二开的 `docs/` 设计文档，包括架构、运行时、翻译流水线、设置存储、划词工具和发布流程。
- README、贡献指南、隐私页、支持页和 Chrome Web Store 发布说明同步更新。
- `.gitignore` 增加本地素材、临时文件、历史备份和个人化输入文件规则。

## v1.0.0 - 2026-05-11

- 第一版可用 Chrome MV3 扩展。
- 支持全文翻译、逐段渲染、目标语言跳过、缓存、失败重试、术语表、自动翻译规则和虚拟滚动续翻译。
- 支持右键菜单、悬浮工具、快捷设置、选中文本 AI 工具和 Markdown 结果渲染。
- 支持通用模型配置、单工具模型覆盖、多模型切换、提示词自定义和工具调试。
- 支持中文为主的多语言界面与 Manifest 国际化。
- 提供产品页、隐私政策、支持页和 Chrome Web Store 发布材料。
