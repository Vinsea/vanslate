✦AI 行为与工具

页面处理提示词 用于全文翻译、自动翻译和虚拟滚动续翻译
作用说明

系统提示词定义模型角色和输出规则；用户提示词负责告诉模型如何处理本次页面文本，并必须保留返回 JSON 的要求，否则页面译文无法逐段写回。

这两段提示词会影响全文翻译的结构化回写协议，当前仅做透明展示，不开放编辑。

页面文本采集规则 控制全文翻译会扫描哪些 DOM 文本块
规则说明

按 JSON 配置标签白名单和评论/反馈区域识别规则。blockTags 是常规正文标签，formTags 是表单说明标签，supplementalTags 会结合 supplementalContextPattern 采集评论等区域，metadataPattern 用于排除作者、日期、评分等元信息。

跳过翻译规则 控制哪些短文本、代码、数字、链接等无需发送给模型
规则说明

按 JSON 配置整段跳过规则。enabled 控制是否启用；patterns 是正则表达式数组，支持普通正则源字符串，也支持 /pattern/flags 写法。仅建议配置高置信规则，避免把需要翻译的正文整段跳过。
